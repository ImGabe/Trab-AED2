# Trabalho 2 de Algoritmo e Estrutura de Dados II

Implementações de algoritmos de Programação Dinâmica, Backtracking e Algoritmos Gulosos na resolução do problema "Número Máximo de Palavras Palíndromas em Texto".

## Número Máximo de Palavras Palíndromas em Texto

Dada uma string, particioná-la em substrings de forma que cada uma seja um palíndromo (lê-se igual de frente para trás e vice-versa), minimizando o número de cortes necessários.
